<h1>StarGame</h1>
<p>Данный проект - игра космическая стрелялка. Необходимо управлять кораблем, уничтожать астероиды и вражеские корабли. На финальном уровне нужно будет сразиться с боссом.</p>
<h2>Описание</h2>
<p>При запуске игры открывается игровое меню, в котором можно:</p>
<ul>
  <li>Начать новую игру</li>
    <li>Ознакомиться с полезной информацией (краткое описание игры и информация об управлении)</li>
    <li>Посмотреть результаты других игроков</li>
    <li>Включить/выключить музыку и звуки</li>
    <li>Выйти из игры</li>
</ul>
<h2>Игровой процесс</h2>
<p>Цель игры - уничтожение астретоидов и вражеских кораблей. При их уничтожении могут выпадать бонусы, необходимые персонажу. Так же сверху передвигается помощник героя - летающая тарелка, 
  которая переодически сбрасывает бонусы. Жизнь и количество патронов ограничены. При столкновении с астероидом жизнь уменьшается. После уничтожения всех астероидов
  и вражеских кораблей игрок переходит на следующий уровень. На последнем уровене нужно будет сразиться с главным боссом.</p>
<h2>Виды бонусов:</h2>
<ol>
  <li>Аптечка</li>
   <li>Запас патронов</li>
   <li>Монеты</li>
</ol>
<p>С помощью монет можно улучшить следующие скилы героя</p>
<ul>
<li>Максимальный уровень жизни</li>
  <li>Текущий уровень жизни</li>
  <li>Вероятность критического удара (урон увеличивается в 3 раза)</li>
  <li>Уменьшить время, через которое помощник сбрасывает бонусы</li>
  <li>Получить более мощное оружие</li>
  <li>Расширить магнитное поле (если бонус попадает в эту область, то он притягивается к кораблю)</li>
</ul>
<h2>Управление</h2>
<ul>
  <li>W или стрелка вверх - вперед</li>
  <li>S или стрелка вниз - назад</li>
  <li>A или стрелка влево - поворот влево</li>
  <li>D или стрелка вправо - поворот вправо</li>
  <li>U - открыть магазин</li>
  <li>Пробел - стрельба</li>
</ul>
